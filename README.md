# Praca-inżynierska
Projekt i budowa systemu do śledzenia pozycji kątowej w czasie rzeczywistym z wykorzystaniem inercyjnych jednostek pomiarowych (IMU) oraz wizualizacji jego ruchu w aplikacji 3D

# Opis repozytorium
Kod do mikrokontrolera jest w pliku Arduino kod.ino

Kod do aplikacji wizualizacyjnej jest w folderze Aplikacja do wizualizacji 3D
